  # Timeslot

## dailyTimePair
It take a object of array, e.g. `[{startTime: [10, 0], endTime: [12, 0]}]` - [hour, min]

{' '}
<DateSlotPicker
  dailyTimePair={[
    {
      startTime: [10, 0],
      endTime: [12, 0],
    },
    {
      startTime: [14, 0],
      endTime: [23, 59],
    },
  ]}
/>

    ``` js filename="example.ts"
    import {DateSlotPicker} from 'react-dateslot-picker';
    import 'react-dateslot-picker/dist/style.css'

    const Example  = () => {

      const props = {
        dailyTimePair: {[{
          startTime: [10, 0],
          endTime: [12, 0]
        }, {
          startTime: [14, 0],
          endTime: [23, 59]
        }]}
      };

      return (
        <DateSlotPicker
          {...props}
        />
      )
    };

```
</Unstyled>
