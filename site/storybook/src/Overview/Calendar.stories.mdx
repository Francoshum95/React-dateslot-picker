  ## disableWeekly

  Disable date weekly 
  Start from 1 to 7, where 1 represents Monday and 7 represents Sunday.

  There are various ways to specify a range of days:

  - You can use a string to represent multiple days, e.g., "1-3", which will disable Monday to Wednesday.
  - You can also use individual numbers to represent specific days.
  - Alternatively, you can use a combination of strings and numbers to specify a range of days.

  <DateSlotPicker
    disableWeekly={["1-3", 4, "5-6"]}
  />

    ``` js filename="example.ts"
    import {DateSlotPicker} from '@react/dateslot-picker';
    import '@react/dateslot-picker/dist/style.css'

    const Example  = () => {

      const props = {
        disableWeekly: {["1-3", 4, "5-6"]}, // from Monday to Suturadqy
      };

      return (
        <DateSlotPicker
          {...props}
        />
      )
    }; 
    ```
</Unstyled>

  ## disableSpecific

  Disable specific date 
  
  There are various ways to specify a range of days:

  - You can use a string to represent multiple days, e.g., "1-3", which will disable 1th to 3rd date.
  - You can also use individual numbers to represent specific days.
  - Alternatively, you can use a combination of strings and numbers to specify a range of days.

  <DateSlotPicker
    disableSpecific={["1-5", 6, 7 , "8-10"]}
  />

    ``` js filename="example.ts"
    import {DateSlotPicker} from '@react/dateslot-picker';
    import '@react/dateslot-picker/dist/style.css'

    const Example  = () => {

      const props = {
        disableSpecific: {["1-5", 6, 7 , "8-10"]}, // from 1st date to 10th date
      };

      return (
        <DateSlotPicker
          {...props}
        />
      )
    }; 
    ```
</Unstyled>
